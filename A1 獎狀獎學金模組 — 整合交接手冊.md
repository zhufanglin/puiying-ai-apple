## 1. 模組概述

本模組是「培英中學 AI 數智化平台 — Apple 子系統」的 **獎狀發放 + 獎學金申請審批** 全流程管理模組。

- **業務頻次**：一年 3 次（上學期 / 中六畢業禮 / 下學期）
- **預期節省**：50-60% 手動工時
- **核心能力**：模板管理 → 獎狀創建 → 獎學金核算 → 確認 → 生成 PDF 證書 → 讀稿製作

## 5. 狀態機

### 5.1 獎狀狀態流轉

```
draft ─┬─→ calculated ──→ confirmed
       │                   
       └─→ confirmed       (從總覽頁快速確認，跳過核算)
       │
       └─→ cancelled       (任意非 cancelled 狀態可取消)
```

### 5.2 獎學金申請狀態

```
pending ──→ approved
        │
        └─→ rejected
```

- `pending → approved/rejected` 不可逆，不可重複審核
- `cancelled` 狀態不可再操作

## 6. 核心業務邏輯

### 6.1 獎學金核算（service.py: `calculate_scholarship`）

- 根據獲獎學生的 **rank（獲獎等級）** 計算每人獎學金金額
- 支援自訂規則（如 `{"一等獎": 1000, "二等獎": 500}`）
- 自訂規則為空時使用默認規則
- 支援繁體/簡體 rank 自動轉換（如 "一等獎" ↔ "一等獎"）
- 未匹配到規則的 rank 使用默認金額（"默認"/"默认"）
- 核算後狀態變更為 `calculated`

### 6.2 證書編號規則

```
{前綴}-{四位序號}
例：MDL-2025-0001、CERT-1-1
```

### 6.3 證書生成（certificate_service.py）

- **DOCX**：使用 python-docx + docxtpl，Jinja2 模板語法
- **PDF**：使用 WeasyPrint（HTML → PDF）
- 批次生成支援多份證書

## 7. 權限列表

| 權限碼 | 說明 | 用途 |
|--------|------|------|
| `AWARDS_READ` | 讀取權限 | GET 類端點 |
| `AWARDS_WRITE` | 寫入權限 | POST/PUT 建立/更新 |
| `AWARDS_DELETE` | 刪除權限 | DELETE 類端點 |
| `AWARDS_APPROVE` | 審核權限 | 獎學金審核 |

權限裝飾器用法：
```python
from app.core.permissions import require_permission, Permissions

@router.post("/{award_id}/calculate")
async def calculate_award_scholarship(
    ...
    user: User = Depends(require_permission(Permissions.AWARDS_WRITE)),
):
```

## 10. 整合注意事項

### 10.1 前端

1. **API 請求核心**：`lib/api.ts` 已處理 token 注入和錯誤解析（含非 JSON 回應和 FastAPI 422 錯誤）
2. **NProgress 路由進度條**：`lib/api.ts` 中整合，所有 API 請求自動顯示載入動畫
3. **共享 UI 元件**：使用 `@/components/ui/PageHeader`、`FormSection`、`DataTable`
4. **TypeScript 類型**：所有類型定義在 `lib/types/awards.ts`，後端 Schema 變更時需同步
5. **Tailwind 顏色**：使用 `primary-400`、`primary-500` 等自定義色板

### 10.2 後端

1. **所有端點均已掛載權限**，整合時無需額外處理
2. **分頁使用 `PageParams` 依賴注入**：`query: AwardQuery = Depends()`
3. **回應使用 `APIResponse` 和 `PaginatedData`**：`app/common/schemas.py`
4. **錯誤使用 `raise_error(*BUSINESS_ERROR, detail={...})`**：`app/common/errors.py`
5. **級聯刪除**：刪除 Award 時自動級聯 AwardRecipients
6. **證書編號唯一性**：`certificate_no` 設有 UNIQUE 約束
